📋 페이지 개요

파일명: admin.html

용도: 플랫폼 전체를 관리하는 시스템 관리자 페이지

사용자 타입: 시스템 관리자 (role: 'admin')

접근 권한: 모든 데이터와 설정에 대한 전체 권한

주요 기능

  • 사용자 관리 모든 사용자 조회, 수정, 삭제, 크레딧 지급
  • 회사 관리 기업 계정 승인, 관리, 통계 확인
  • AI 에이전트 관리 프롬프트 수정, 모델 변경, 비활성화
  • 크레딧 관리 패키지 생성, 수정, 결제 현황 모니터링
  • 시스템 모니터링 로그 확인, 통계 분석, 시스템 설정

🎨 UI 컴포넌트 정의

사이드바 네비게이션

  • 대시보드 전체 플랫폼 현황 요약 (showSection('dashboard'))
  • 사용자 관리 사용자 목록, 검색, 필터링 (showSection('users'))
  • 회사 관리 기업 계정 관리 (showSection('companies'))
  • AI 에이전트 에이전트 설정 관리 (showSection('agents'))
  • 크레딧 관리 패키지, 결제 관리 (showSection('credits'))
  • 로그/통계 시스템 로그, 사용 통계 (showSection('logs'))
  • 시스템 설정 API 키, 시스템 설정 (showSection('settings'))

  • 사용자 검색/필터 이름, 이메일, 계정타입별 필터링
  • 정보 수정 모달 사용자 정보 인라인 편집
  • 크레딧 지급 수량 입력 후 즉시 반영
  • 계정 정지/복원 확인 메시지 후 상태 변경

AI 에이전트 관리

  • 프롬프트 편집 텍스트 에디터로 프롬프트 수정
  • 모델 변경 드롭다운으로 AI 모델 선택
  • 에이전트 활성화/비활성화 토글 버튼으로 상태 제어

시스템 모니터링

  • 실시간 로그 자동 새로고침, 필터링 기능
  • 통계 차트 Chart.js로 사용량 그래프 표시
  • 알림 시스템 에러, 경고 발생 시 실시간 알림

🔒 보안 및 권한

접근 제어

  • 로그인 상태 확인 페이지 로드 시 인증 상태 검증
  • 관리자 권한 확인 role='admin' 체크, 권한 없으면 리다이렉트
  • 중요 작업 확인 삭제, 정지 등 민감한 작업은 2차 확인

데이터 보호

  • API 키 마스킹 설정에서 API 키는 마스킹 처리
  • 개인정보 처리 사용자 정보 조회/수정 시 로그 기록
  • 세션 관리 일정 시간 미사용 시 자동 로그아웃
