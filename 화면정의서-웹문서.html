📋 프로젝트 개요

🎯 프로젝트 목표

다양한 AI 에이전트를 통합 관리하고 활용할 수 있는 실제 테스트 가능한 플랫폼 구현

✨ 주요 특징

  • 실제 회원가입/로그인 기능
  • 통합된 데이터 관리 시스템
  • 관리자/사용자 페이지 연동
  • Vercel 배포 지원

🤖 AI 에이전트

  • 일반사무: 4개 에이전트
  • 마케팅/광고: 4개 에이전트
  • 콘텐츠 제작: 2개 에이전트

👥 사용자 역할

  • 일반 사용자 (개인/회사)
  • 회사 관리자
  • 시스템 관리자

🏗️ 시스템 아키텍처

기술 스택

HTML5 CSS3 JavaScript ES6+ LocalStorage Font Awesome 6.0 Vercel

전체 구조

AI Agent Hub Platform ├── 메인 사용자 페이지 (index.html) ├── 시스템 관리자 페이지 (admin.html) ├── 회사 관리자 페이지 (company-admin.html) └── 공통 데이터 관리 시스템 (data-manager.js)

📊 데이터 구조 정의

User 객체

{ id: string, // 고유 식별자 email: string, // 이메일 (로그인 ID) password: string, // 비밀번호 (해시화 필요) name: string, // 사용자명 role: string, // 'admin' | 'company_admin' | 'user' accountType: string, // 'admin' | 'company' | 'individual' credits: number, // 보유 크레딧 createdAt: string, // 가입일 (ISO 8601) status: string, // 'active' | 'inactive' | 'suspended' // 회사 계정 전용 필드 companyName?: string, businessNumber?: string, department?: string, position?: string }

Agent 객체

{ id: string, // 에이전트 고유 ID title: string, // 에이전트 제목 description: string, // 설명 icon: string, // Font Awesome 아이콘 클래스 credits: number, // 필요 크레딧 category: string, // '일반사무' | '마케팅/광고' | '콘텐츠 제작' status: string, // 'active' | 'inactive' createdAt: string, // 생성일 usage: number // 총 사용 횟수 }

Usage Record 객체

{ id: string, // 사용 기록 ID userId: string, // 사용자 ID agentId: string, // 에이전트 ID credits: number, // 사용된 크레딧 timestamp: string, // 사용 시간 result?: object // 결과 데이터 (선택적) }

관리자 로그인 → 관리자 대시보드 → 데이터 관리 → 통계 확인 → 시스템 설정

  3. 실행 버튼 클릭
  4. 크레딧 차감

📊 결과 확인

  1. 로딩 상태 표시
  2. 시뮬레이션 결과 생성
  3. 결과 페이지 표시
  4. 다운로드/재실행 옵션

💳 크레딧 관리

  1. 잔액 부족 시 충전 페이지 이동
  2. 패키지 선택
  3. 결제 수단 선택
  4. 결제 완료 후 크레딧 충전

🔗 API 요구사항 (백엔드 연동 시)

인증 API

POST /api/auth/register # 회원가입 POST /api/auth/login # 로그인 POST /api/auth/logout # 로그아웃 GET /api/auth/profile # 프로필 조회 PUT /api/auth/profile # 프로필 수정

사용자 관리 API

GET /api/users # 사용자 목록 (관리자) GET /api/users/:id # 사용자 상세 PUT /api/users/:id # 사용자 정보 수정 DELETE /api/users/:id # 사용자 삭제 POST /api/users/:id/credits # 크레딧 충전/차감

에이전트 API

GET /api/agents # 에이전트 목록 GET /api/agents/:id # 에이전트 상세 POST /api/agents/:id/run # 에이전트 실행 GET /api/agents/results/:id # 결과 조회

통계 API

GET /api/stats/dashboard # 대시보드 통계 GET /api/stats/usage # 사용량 통계 GET /api/stats/revenue # 매출 통계

기능 테스트

🔐 회원가입/로그인

  1. 개인 계정 회원가입 → 로그인 → 대시보드 접근
  2. 회사 계정 회원가입 → 추가 정보 입력 → 승인 대기
  3. 잘못된 정보 입력 시 오류 처리
  4. 비밀번호 확인 불일치 시 경고

🤖 AI 에이전트 실행

  1. 충분한 크레딧으로 에이전트 실행 → 성공
  2. 부족한 크레딧으로 실행 시도 → 충전 페이지 리다이렉트
  3. 결과 생성 → 다운로드 → 재실행
  4. 10개 에이전트 모두 실행 테스트

👥 관리자 기능

  1. 관리자 로그인 → 회원 목록 조회 → 정보 수정
  2. 에이전트 상태 변경 → 실시간 반영 확인
  3. 통계 데이터 정확성 검증
  4. 권한 기반 접근 제어 확인

💳 크레딧 시스템

  1. 크레딧 충전 프로세스 테스트
  2. 사용량 차감 정확성 확인
  3. 잔액 부족 시 알림 동작
  4. 사용 내역 기록 확인
