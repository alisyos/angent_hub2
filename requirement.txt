2.1.1 회원가입 기능

#. 회원 유형 분류
•	일반 계정: 개인 사용자 계정
•	회사 계정: 기업 사용자 계정 (회사명, 사업자번호 등 추가 정보 필요)

#. 회사 계정 세부 분류
•	회사 관리자 계정: 해당 회사의 사용자 관리 권한 보유
•	회사 일반 계정: 일반 직원 계정, 회사 관리자가 초대/관리
2.1.2 로그인 기능
#. 로그인 방식
•	이메일/비밀번호 로그인
•	소셜 로그인: Google, 네이버, 카카오
#. 추가 보안 기능
•	비밀번호 찾기/재설정
•	로그인 시도 제한: 5회 실패 시 계정 잠금
•	세션 관리: 자동 로그아웃

#. 개인정보 관리
•	기본 정보 수정 (이름, 이메일, 연락처)
•	비밀번호 변경
•	프로필 이미지 업로드
#. 회사 계정 추가 정보
•	회사명, 부서, 직급
•	사업자번호 (회사 관리자만)
•	회사 로고 업로드

2.2.3 에이전트별 크레딧 정책

•	복잡도에 따른 차등 크레딧 적용
•	정상적인 결과물 출력 완료 시 크레딧 차감
•	에러 발생 시 크레딧 차감 없음
#. 크레딧 관리
•	실시간 잔액 확인
•	사용 내역 상세 조회
•	크레딧 유효기간 관리 (선택사항)
